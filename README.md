 **HAMMERTIME** — _F1 Race Outcome Simulator v1.0_
Monte Carlo + ML hybrid simulator for Formula 1 race predictions.

Features:
- 10,000-iteration Monte Carlo simulation engine
- XGBoost ML model for lap time prediction
- Live data from FastF1, Jolpica API, and OpenF1
- 20 drivers with real headshot photos
- 24 circuits with SC/rain probabilities
- Next Race countdown with live timer
- Recent race winners with podium display
- Full season schedule calendar
- Championship standings (drivers + constructors)
- Interactive charts: lap times, tyre heatmap, DNF gauges, radar
- Pre-race AI predictions with PNG export
- Premium dark theme with glassmorphism design

Stack: FastAPI, XGBoost, NumPy, Chart.js, Vanilla JS
